Ubuntu下重置忘记的MySql root密码

有时，我们可能会忘记mysql root密码，这时我们就需要重置它。
下面就讲下具体操作:

  1. 停止mysql进程 或者
  2. 用–skip-grant-tables选项启动mysql进程
  3. 启动mysql客户端
  4. 刷新MySQL的系统权限相关表，重置密码
  5. 刷新MySQL的系统权限相关表，重启mysql进程

参考：Reset lost/forgot MYSQL root password ubuntu
